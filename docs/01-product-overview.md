# Product Overview

## 1. Concept

**Fan League** is an internal web application for colleagues to predict FIFA World Cup 2026 match scores. 

The game is 100% free with no entry fees, wagers, financial stakes, or real-money prizes. Users submit predictions for each match. At the end of the tournament, the system aggregates points and awards cosmetic ranks and fun badges on the leaderboard.

The application has zero betting mechanics, wagers, odds, or stakes. It is strictly an internal colleague-engagement mini-game.

The app supports multi-tenant company configurations. Each user belongs to a specific company/league and is identified by a unique nickname + passcode issued by the league organizer/admin.

Roles:
- `PLAYER`: Predict match scores, view leaderboards, unlock fun badges, and view game rules.
- `ADMIN`: Manage a single company/league.
- `SUPER_ADMIN`: Create and manage company entities/leagues, and bootstrap initial company administrators.

## 2. Objectives

- Promote fun colleague interaction during the World Cup 2026.
- Provide a simple, fast prediction interface.
- Ensure transparency of scorelines, predictions, and rankings.
- Simplify administration for league organizers.
- Deploy quickly using a modern TypeScript stack.
- Deliver a clean, professional UI optimized for mobile and desktop screens.

## 3. User Roles & Capabilities

### Players
- Log in using a passcode generated by the organizer.
- Passwordless and email-free authentication; identified solely by nickname.
- View match fixtures.
- Submit score predictions.
- View points, rankings, badges, and predictions of other players (once the match is locked).

### Admin / Organizers
- Manage their company/league entity.
- Register participants and view/reset their passcodes.
- Configure match prediction status (open or close).
- Set match prediction lock windows (defaults to kickoff - 5m).
- Log final match scores or trigger automatic sync.
- Trigger automatic score calculations and leaderboard updates.
- Export prediction datasets, leaderboard standings, and audit logs.

## 4. MVP Scope

1. Passcode-based login.
2. Match schedule and status.
3. Score prediction submissions.
4. Auto-locking predictions before kickoff.
5. Administrative manual score entry.
6. Automatic point calculation engine.
7. Dynamic leaderboard standings.
8. Cosmetic badges and achievements tracker.
9. Match detail screen displaying all predictions once locked.
10. Admin toggle for match prediction availability.
11. CSV/XLSX exports for core data.

## 5. Out of Scope for MVP

- Detailed live match timelines.
- Real-time team lineups.
- Push notifications.
- Integrated chat rooms or commenting.
- Financial transactions or payment gateways.
- Multi-season histories.

## 6. Product Principles

- Simplicity over complexity.
- Transparency over decoration.
- Clear rules over feature bloat.
- Speed: Prediction submission in under 10 seconds.
- Clean design: Visual elements must never resemble a betting/casino application.
